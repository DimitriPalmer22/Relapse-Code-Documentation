# The Player

Right now, the `PlayerInfo` script contains all the data for the player as well as the logic.

## Things that Can Be Put into a Scriptable Object

### Basic Player Info
- Current / Max Health
- Current / Max Toxicity
- Remaining Relapse Time
- Current / Max Stamina

### Power Info
- Current Power Loadout
- Power Tokens
