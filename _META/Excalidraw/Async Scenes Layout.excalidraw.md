Level Section 1

The static & non-interactable game
objects found in Level Section 1.

(Walls, floors, ceilings) ^mv8Mv8jH

These are the things that constantly need to stay in the game's memory for as long as the player is in the level. ^OoWpk1oT

These are the things that are specific to one part of the level. We don't want them to ALWAYS be active, but we don't want them to be completely unloaded either.

For example, if I 
- put a pickup in the non-persistent sub-scene
- grabbed the pickup in-game
- unloaded this portion of the level
- and then reloaded this portion of the level

The pickup would respawn! We don't want this because it would be easy to exploit the game this way. ^yojBo6ll
